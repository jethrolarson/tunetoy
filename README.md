Getting Started
===============

[Coffee-script][1]
------------------

You said you wanted to go deep, well here we go:

1. [Install node][2]

2. [Install npm][3]

3. `sudo npm install coffee-script`

4. Run the watch script

  `coffee -o static/js/ --watch coffee/*.coffee`

5. [tmbundle](5)

[Stylus][4]
-----------

1. `sudo npm install stylus`
2. `stylus --watch stylus/ --out static/css`
3. [tmbundle][6]
